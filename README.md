# Tiago Braga | Website Oficial

Website oficial de Tiago Braga, destacando sua atuação como Músico, Escritor e Presidente da OMB-RJ. O projeto apresenta uma landing page premium e responsiva, desenvolvida em HTML, CSS e JS, com foco em estética minimalista e alta performance. Explora os pilares de Consciência, Propósito e Impacto que regem sua carreira e liderança institucional.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e SEO.
- **CSS3**: Design moderno, responsivo e animações customizadas.
- **JavaScript (Vanilla)**: Interatividade e efeitos de scroll.
- **Lucide Icons**: Ícones minimalistas.
- **Cloudflare Pages**: Hospedagem e performance global.

## 🛠️ Como rodar o projeto localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/AlexanderBits/Cantor-tiago-braga.git
   ```
2. Abra o arquivo `index.html` no seu navegador.
